# jdex
Apk disassembler and decompiler based on [jadx](https://github.com/skylot/jadx)

# Building
* Requires java 21

```sh 
./gradlew shadowJar
```
```sh 
java -jar app/build/libs/jdex.jar ~/Downloads/pinning-demo.apk
```
