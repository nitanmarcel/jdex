# jdex
Apk disassembler and decompiler based on [jadx](https://github.com/skylot/jadx)

![Alt text](public/jdex.png)

# Building
* Requires Java 21
* Requires CMake

```sh
git submodule update --init
./gradlew shadowJar
```
```sh
java -jar app/build/libs/jdex.jar
```
